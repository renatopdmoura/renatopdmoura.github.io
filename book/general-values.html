CSS Book- Valores gerais

Valores gerais são valores que estão dísponiveis em varias propriedades. Um exemplo desses valores são initial e inherit.

  • initial: define que para determinada propriedade, o elemento exibirá o comportamento padrão.
  • inherit: define que para determinada propriedade, o elemento exibirá o comportamento herdado do elemento que o contém.
