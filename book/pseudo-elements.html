Pseudo Elementos

Pseudoelementos são para estilizar uma parte de uma tag(como estilizar a primeira letra de um paragráfo por exemplo). Os pseudoelementos são acessados pela seguinte regra sintática:

                        selector::pseduo-element{
                            property: value;
                        }

Os pseudoelementos disponíveis são os seguintes:

  1. first-line
  2. first-letter
  3. before
  4. after
  5. marker
  6. selection
