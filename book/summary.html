CSS Book- Sumário

Motivação

CSS é um conjunto de propriedades que definem a estilização de tags html. Em contrapartida ao antigo método de formatação de tags(como no html4), css define uma folha de estilos útil para ser compartilhada por diferentes páginas de forma organizada e eficiente. Este livro tem como objetivo servir de guia para o desenvolvedor que deseja obter uma noção inicial e essêncial dessa tecnologia.
