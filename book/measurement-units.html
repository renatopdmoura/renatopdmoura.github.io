Unidades absolutas

Unidades absolutas são independetnes de resolução. São direcionadas ao uso de páginas para impressão ou para dispositivos cuja resolução de saída seja conhecida.

  • cm: corresponde a centimetros.
  • mm: corresponde a milimetros.
  • in: corresponde a polegadas onde cada polegada é igual à 96 que são iguais a 2.54cm.
  • px: corresponde a pixels onde cada pixel é igual à 1/96 de uma polegada.
  • pc: corresponde a picas onde cada pica é igual à 12 pontos.

Unidades relativas
