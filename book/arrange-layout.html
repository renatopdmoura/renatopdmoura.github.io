Dispondo elementos

A seguir lista-se propriedades que organizam o layout do seu site:

  1. position
  2. float
  3. z-index

position

A propriedade position define a posição de um elemento na página. Os valores para a propriedade são:

  • sticky

      • Ao definir sticky para propriedade position, ao rolar a página, o elemento se prenderá ao topo/inferior(para rolagem vertical, ou no lado esquerdo/direito para rolagem horizontal).

                                          .pos-debug-father{
                                        text-align: center;
                                        background-color: antiquewhite;
                                        box-sizing: border-box;
                                        max-width: 100%;
                                        height: auto;
                                        padding: 20px;
                                        border: 2px solid #000;
                                        border-radius: 5px;
                                        margin: auto;
                                        margin-top: 20px;
                                        margin-bottom: 20px;
                                        position: sticky;
                                        top: 0;
                                    }
                                    .pos-debug-children{
                                        text-align: center;
                                        background-color: aliceblue;
                                        box-sizing: border-box;
                                        max-width: 20%;
                                        height: auto;
                                        padding: 20px;
                                        border: 2px solid #000;
                                        border-radius: 5px;
                                    }
  • static

      • Ao definir static para propriedade position de um elemento, o elemento não apresentará qualquer comportamento especial e sua posição permanecerá a mesma.

                                          .pos-debug-father{
                                        text-align: center;
                                        background-color: antiquewhite;
                                        box-sizing: border-box;
                                        max-width: 100%;
                                        height: auto;
                                        padding: 20px;
                                        border: 2px solid #000;
                                        border-radius: 5px;
                                        margin: auto;
                                        margin-top: 20px;
                                        margin-bottom: 20px;
                                        position: static;
                                        left: 100px;
                                    }
                                    .pos-debug-children{
                                        text-align: center;
                                        background-color: aliceblue;
                                        box-sizing: border-box;
                                        max-width: 20%;
                                        height: auto;
                                        padding: 20px;
                                        border: 2px solid #000;
                                        border-radius: 5px;
                                    }

      Note como um deslocamento para esquerda de 100 pixels(left: 100px) resulta em um noup.

  • absolute

      • Ao definir absolute para propriedade position, o elemento terá sua posição definida em função de seu elemento ancestral direto e caso não houver, a tah <body> será usada como referência.

                                          .pos-debug-father{
                                        text-align: center;
                                        background-color: antiquewhite;
                                        box-sizing: border-box;
                                        max-width: 100%;
                                        height: auto;
                                        padding: 20px;
                                        border: 2px solid #000;
                                        border-radius: 5px;
                                        margin: auto;
                                        margin-top: 20px;
                                        margin-bottom: 20px;
                                        position: relative;
                                    }
                                    .pos-debug-children{
                                        text-align: center;
                                        background-color: aliceblue;
                                        box-sizing: border-box;
                                        max-width: 20%;
                                        height: auto;
                                        padding: 20px;
                                        border: 2px solid #000;
                                        border-radius: 5px;
                                        position: absolute;
                                        top: -2px;
                                        left: -2px;
                                    }
  • relative

      • Ao definir relative para propriedade position, o elemento terá sua posição relativa a posição no fluxo de dados da página na qual fora definido.

                                          .pos-debug-father{
                                        text-align: center;
                                        background-color: antiquewhite;
                                        box-sizing: border-box;
                                        max-width: 100%;
                                        height: auto;
                                        padding: 20px;
                                        border: 2px solid #000;
                                        border-radius: 5px;
                                        margin: auto;
                                        margin-top: 20px;
                                        margin-bottom: 20px;
                                        position: relative;
                                        left: 20px;
                                    }
                                    .pos-debug-children{
                                        text-align: center;
                                        background-color: aliceblue;
                                        box-sizing: border-box;
                                        max-width: 20%;
                                        height: auto;
                                        padding: 20px;
                                        border: 2px solid #000;
                                        border-radius: 5px;
                                    }
  • fixed

      • Ao definir fixed para propriedade position, o elemento manterá sua posição fixa independente da rolagem página.

                                          .pos-debug-father{
                                        text-align: center;
                                        background-color: antiquewhite;
                                        box-sizing: border-box;
                                        max-width: 100%;
                                        height: auto;
                                        padding: 20px;
                                        border: 2px solid #000;
                                        border-radius: 5px;
                                        margin: auto;
                                        margin-top: 20px;
                                        margin-bottom: 20px;
                                        position: fixed;
                                        top: 0;
                                    }
                                    .pos-debug-children{
                                        text-align: center;
                                        background-color: aliceblue;
                                        box-sizing: border-box;
                                        max-width: 20%;
                                        height: auto;
                                        padding: 20px;
                                        border: 2px solid #000;
                                        border-radius: 5px;
                                    }

As propriedades que determinam as posições explicitamente de um elemento são:

  • top
  • left
  • right
  • bottom

Essas propriedades se comportaram diferentemente para cada valor de position pois cada valor de position define um referêncial distinto. As unidades de medidas aceitas para cada valor são descritas no capítulo Unidades de Medidas.

float

Definir a propriedade float para um elemento determina que esse elemento deve flutuar de modo que o conteúdo que o procede ocupe dinâmicamente o espaço disponível.

Flutuações podem ocorrer em suma para dois sentidos:

  • right: o elemento flutuará para direita.
  • left: o elemento flutuará para esquerda.

clear

Determina como um elemento adjacente a um elemente flutuante deve se comportar. clear pode receber os seguintes valores:

  • none: o elemento adjacente não será empurrado para a esquerda ou direita do elemento flutuante.
  • left: o elemento é empurrado abaixo dos elementos flutuantes à esquerda.
  • right: o elemento é empurrado abaixo dos elementos flutuantes à direita.
  • both: o elemento é empurrado abaixo dos elementos flutuantes à esquerda e direita.
  • inherit: herda o valor de seu elemento pai.

clear deve ser definida sempre após uma propriedade float .

z-index

A proprieda z-index resolve o problema de sobreposição de elementos. Por padrão, o elemento com maior prioridade visual é aquele definido por último(como uma pilha de livros, o ultimo elemento é(idealmente) o primeiro a ser avaliado quando desejamos manipular a pilha). z-index possui valores os quais em ordem crescente representa a prioridade de sobreposição na página. Por exemplo, se definirmos para uma divA z-index: 1 e para uma divB z-index: 2 a divA se sobreporá a divB uma vez que dividem o mesmo ou parte do mesmo espaço no documento.
