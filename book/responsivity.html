Consultas de Mídia

Uma regra @media permite definir diferentes formatações para diferentes mídias. Usando a regra @media podemos por exemplo definir um padrão de cores diferentes para quando o usuário abre o seu site em um dispositivo mobile ou no computador como também carregar layouts diferentes para seu site dependendo do dispositivo em que ele é acessado. Traduzindo o termo "media query" obtemos um significado claro e objetivo do que tal regra é capaz de realizar, ou seja consultas de mídia. As consultas bastante comuns de serem realizadas são, não limitadas à:

  • verificar a largura e altura da viewport.
  • verificar a largura e altura do dispositivo.
  • verificar o modo de orientação de um dispositivo.
  • verificar a resolução atual.

Dado tal, por pordermos verificar caracteristicas dos dispositivos em que o site opera, @media é fundamental quando tocamos no assunto de responsividade. Vamos verificar um exemplo abaixo:

                        @media screen and (max-width: 600px){
                            body{
                                background-color: #fdd;
                            }
                        }
                        @media screen and (min-width: 600px){
                            body{
                                background-color: #ddd;
                            }
                        }

Nesse exemplo, definimos duas regras especificando a mídia como screen e especificamos uma condição para regra: caso a largura máxima da tela do dispositivo seja 600px. Em caso verdadeiro, definimos o plano de fundo da tag body para um tom rosado. A segunda regra @media é similiar, contudo a condição do lado direito do operador relacional "and" define que a regra deve ser aplicada sempre que o tamanho mínimo da largura do dispositivo sejá igual a 600px.

Com uma pequena ideia de como uma @media funciona, vamos explorar sua definição sintática:

@media mediatype and|not|only (expressions)

Onde:

  • @media: declarado ao definir uma regra de consulta de mídia.
  • mediatype: a variável de mídia a ser verificada, sendo:
    • all: não discrimina entre dispositivos. O padrão.
    • print: define que as consultas serão válidas para impressoras.
    • screen: define que as consultas serão válidas para telas de computadores, smart-phones, tablets...
    • speech: define que as consultas serão válidas para leitores de tela.
  • and | not | only: definem operadores relacionais. Para mais informações sobre consulte Apêndice A: Operadores Relacionais.
  • expressions: uma ou mais expressões condicionais ligadas a consulta de mídias.

    • Existem diversas expressões em CSS. As expressões mais notórias são discutidas a seguir:

    Uma página responsiva construida usando consultas de mídia pode ser visualizada no seguinte link.

    max-width

    Uma expressão que retorna o tamanho máximo da janela de exibição do navegador.

    min-width

    Uma expressão que retorno o tamanho mínimo da janela de exibição do navegador.

    orientation

    Uma expressão que verifica a orientação do dispositivo, se em modo paisagem ou em modo retrato.

    resoltution

    Uma expressão que verifica a resolução do dispositivo de saída.

    O exemplo a seguir mostra como encadear consultas de mídia:

                           @media screen and (max-width: 1280){
                           @media screen and (orientation: landscape){
                               body{
                                   background-color: antiquewhite;
                               }
                           }
                       }

    O trecho de código a seguir define que para toda janela de visualização cuja largura máxima seja igual à 1280 pixels e a orientação do dispositivo seja em modo paisagem, o corpo da página terá uma cor com tom de branco envelhecido.
