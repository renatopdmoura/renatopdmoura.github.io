Sobre Fontes

font-family

A propriedade font-family define a fámilia da fonte do texto. Fontes podem ser distinguinda entre três grupos:

  • Serif
  • Sans-Serif
  • Monospace
  • Cursive
  • Fantasy

Serif

Fontes serifadas apresentam um detalhe em suas extremidades que lhes conferem um visual elegante.

Sans-Serif

Fontes sem serifa possuem um visual limpo o que retrata um visual minimalista.

Monospace

Fontes monoespaçadas apresentam o mesmo espaçamento entre caracteres. Essa caracteristica lhes da um aspecto mecânico.

Cursive

Fontes cursivas imitam a calegrafia humana.

Fantasy

Fontes de fantasia trazem um toque do épico com tendência representativa ao que campo lúdico(filmes, séries, histórias, jogos...).

font-size

A propriedade font-size denota o tamanho da fonte. O tamanho pode ser dado em diferentes unidades como cm, px, em...

Sobre Alinhamento

text-align

A propriedade text-alig define como um texto será alinhado ao bloco que pertence. Podendo ser:

  • left: o texto será alinhado a esquerda.
  • right: o texto será alinhado a direita.
  • center: o texto será alinhado ao centro.
  • justify: o texto será ajustado para que ocupe todo o espaço dentro de uma linha sempre que for possível.

text-align-last

A propriedade text-align-last define como a última linha de um paragrafo deve ser alinhada. Seus valores são tais como em text-align. Exemplificando(à esquerda sem alinhar a última linha e a direita com alinhamento de última linha).

Essa é uma propriedade que deve ser usada em conjunto com white-space e overflow(ainda não vista nesse capítulo). text-overflow é útil pois define como tratar textos que "estouram" o bloco que os contém. O tratamento pode ser feito das seguintes formas:

Tratando fontes de forma responsiva

Uma tecnica bastante simples de fazer com que a fonte de seus textos se redimensione conforme a janela de visualização é definir a propriedade font-size no seletor de body com o valor de 100% enquanto define a medida do tamanho da fonte em seus parágafros usando a unidade em. Como segue:

                        body{
                            font-size: 100%;
                        }
                        p{
                            font-size: 1em;
                        }

1em equivale à 1/16 pontos. Na verdade em usa a DPI disponível pela resolução nativa do monitor. Comumente, monitores de alta DPI usam o equivalente a 16 pixels por polegada(dai a regra 1/16). Em monitores de baixa DPI, essa regra pode mudar, porém não é algo que precise de atenção por parte do desenvolvedor.

Dicas

Fontes encadeadas

Por vezes, ao definir uma família de fontes muito especifica para a propriedade font-family não há garantias de que estas fontes estejam dispoíveis na máquina do usuário. Para diminuir o impacto desse problema, a propriedade font-family aceita uma definição encadeada de nomes de fontes. Isso é útil já que pode ser usada uma fonte similiar sempre que a primeira fonte na cadeia(lida da esquerda para direita) esteja indísponivel. Veja o exemplo no trecho de código a seguir:

                        P{
                            font-family: 'Arial' sans serif, 'Verdana' sans-serif;
                        }

Eventualmente, usar fontes alternativas para substituir uma fonte indisponível por vezes pode resultar em uma fonte substituta com uma proporção maior que a desejada. Para manter a proporção usamos a propriedadefont-size-adjust que obtém o valor de aspecto - calculado pela diferença de tamanho entre a letra minúscula "x" e maiúscula "X" da primeira fonte indicada.

Os valores para font-size-adjust são:

  • none: sem auto ajuste.
  • number: um valor propocional especificado.
  • initial: o valor inicial.
  • inherit: o valor herdado.

Importando fontes

A regra CSS @font-face permite que você importe suas própias fontes diretamente do seu diretório de arquivos. O uso de @font-face deve ser tal como segue:

                        @font-face{
                            font-family: 'Maiandra';
                            src: url(fonts/Maiandra.TTF);
                        }

Uma vez que font-family torna-se um nome de referência arbitrário dado a fonte importada enquanto src: url(myfont) espera o caminho para importação da fonte. Lembre-se de dar preferência a caminhos relativos ao invés de caminhos absolutos pois(muito provavelmente) seu arquivos serão armazendas em uma máquina externa.

Após definir uma regra @font-face, para usar a fonte importada basta especificar o nome de referência dado a fonte importada como valor da propriedade font-family de um elemento:

                        p{
                            font-family: 'Maiandra';
                            font-szie: 2em;
                            text-align: center;
                        }
