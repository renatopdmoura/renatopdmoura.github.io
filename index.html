O que é

ImCreate é um conjunto de arquivos programados em C++ que fornecem uma série de recursos pertencentes ao pipeline de renderização gráfica. Definem as seguintes caracteríticas:

  1. Gerenciador de sombreadores

      2. Funcionalidade atribuida a classe SRW, nela funções como geração de sombreadores e objetos de programa além da inicialização de dados uniformes são realizadas. SRW funciona como classe base para a grande maioria das demais classes.

  2. Observador virtual

      3. Um observador virtual é responsável por determinar a porção da cena vísivel para renderização. Um observador virtual é definido por um instância da classe Camera. Atualmente, o suporte para observadores virtuais é limitado ao modelo First Personal Shooter(FPS).

  3. Carregador de modelos

      4. A classe Object é responsável por realizar o carregamento de modelos 3D a partir da interpretação de arquivos .obj. É classe derivada de SRW, carregamentos texturas e/ou definição de cores uniformes são feitas a partir das chamadas as funções de inicialização de uniformes da classe base.

  4. Carregador de terrenos

      5. Classe Terrain, terrenos são carregados a partir de mapas de altura em formato ".raw" - como no photoshop, ou ".r8" - alternativamente do krita. É classe derivada de SRW, uniformes são inicializados a partir de chamadas as funções da classe base.

  5. Interface de Usuário

      6. Dispõe de classes que descrevem painés, checkbox e texto. Instâncias dos tipos de widgets mantém um relação semântica herdada.

      Classe Panel é especializada em criar painés que serviram de contâiner para outros objetos de Interface gráfica.

      Class Checkbox faz parte do conjunto de botões disponibilizados pela biblioteca e permite criar relações de grupo para um conjunto de caixas de seleção.

      Classe Text usa FreeType como núcleo de funcionalidade. Permite carregamento de texto de forma otimizada usando um único identificador de texturas, ou métodos que carregam caractéres em tempo de execução - como comandos de console e mensagens de chat.

  6. Áudio 3D

      7. A classe Audio é um wrapper para as chamadas da API OpenAL permitindo reproduzir efeitos espaciais.

  7. Banco de Dados

      8. SQLite3 é usado para flexibilizar o gerenciamento de dados para programador garantindo a consitência das regras de ciência de dados imposta pelo *padrão SQL. Bancos de dados são gerados por instânciação de objetos da classe Database.

Tecnologias

  • SDL
  • OpenGL
  • FreeType
  • OpenAL
  • SQLite3
